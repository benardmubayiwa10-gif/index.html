\#  **African Food Survey Analytics Dashboard**



A visually rich, interactive analytics dashboard built with \*\*HTML and CSS\*\* to present insights from an African traditional food market survey.  

The project showcases consumer preferences, demographics, pricing sensitivity, and strategic insights for launching or improving an African restaurant concept.



---



\##  Project Overview



This dashboard summarizes data collected from \*\*1,000 survey respondents\*\* and transforms it into clear, engaging visual insights.  

It is designed for \*\*market research, business strategy, and data storytelling\*\* purposes.



The dashboard includes:

\- Demographic analysis

\- Customer segmentation

\- Menu preferences

\- Pricing willingness

\- Strategic recommendations
